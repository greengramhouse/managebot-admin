วิธีการเชื่อม Rich Menu (สลับหน้า)
  1. คุณต้องสร้าง Rich Menu อย่างน้อย 2 ตัว (เช่น Menu A และ Menu B)
  2. ตั้งชื่อ Alias ID ให้กับ Menu B (เช่น menu-b) ด้านล่างนี้
  3. ใน JSON ของ Menu A ตรงพื้นที่ที่ต้องการให้คลิกแล้วเปลี่ยนหน้า ให้ใส่โค้ด Action แบบนี้: 
"action": {
  "type": "richmenuswitch",
  "richMenuAliasId": "menu-b",
  "data": "user-switched-to-b"
}
